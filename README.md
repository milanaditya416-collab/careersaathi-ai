# 🏆 CareerSaathi AI — Apna Career, Apni Bhasha

**CareerSaathi AI** is India's first Hindi-first AI career platform — built for the 20 Crore+ Hindi-medium students who are locked out of English-only career tools. Free. Mobile-first. Built for Bharat.

## 🎯 The Problem

- **65%** of Indian students from Hindi/regional-medium backgrounds are locked out of English-only career platforms
- **20 Crore+** students have zero access to Hindi career guidance or AI-powered tools
- **₹0** free Hindi AI career counselling exists at scale in India today

---

## ✨ Features

| Tool | What It Does |
|---|---|
| 🏅 **AI Candidate Ranker** | Paste a Job Description → rank multiple candidates by fit score, with hire/pass recommendation |
| 📄 **Resume Analyzer** | Paste your resume → get ATS score, strengths, missing keywords, and a rewritten summary |
| 📝 **JD Generator** | Fill in role details → get a complete, professional job description in seconds |
| 🎤 **Interview Coach** | Practice mock interviews by role & level → get scored feedback + model answers |
| 💰 **Salary Insights** | Enter role, city, experience → get min/avg/max LPA, top skills, and negotiation tips |
| 🤖 **Hinglish Chat** | Ask anything about your career in Hindi or English — CareerSaathi replies in Hinglish |

## 🛠️ Tech Stack

- **Frontend** — React + TanStack Router, Tailwind CSS, Framer Motion
- **AI** — Google Gemini (via Lovable AI Gateway), Vercel AI SDK
- **Validation** — Zod (end-to-end type-safe AI inputs/outputs)
- **Build** — Vite + Bun
- **Platform** — Built & deployed on [Lovable](https://lovable.dev)

## 🏗️ Run Locally

```bash
# Clone the repo
git clone https://github.com/milanaditya416-collab/careersaathi-ai.git
cd careersaathi-ai

# Install dependencies
bun install

# Start dev server
bun run dev
```

> Requires a `LOVABLE_API_KEY` environment variable for AI features.
